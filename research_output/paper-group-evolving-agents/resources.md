## Related Papers

### Papers This Work Builds On (Read First)

| Paper | Why Relevant |
|-------|-------------|
| Darwin Gödel Machine (DGM) | Direct predecessor; GEA's main baseline — tree-structured individual evolution |
| LIVE-SWE-AGENT | Self-evolving agents on SWE-bench (arXiv 2511.13646) |
| Reflexion | Retrospective feedback for agent improvement — foundational prior work |
| Quality-Diversity (QD) Optimization | Theoretical basis for performance-novelty selection criterion |
| Gödel Machine (Schmidhuber) | Original self-rewriting agent concept |

### Related / Competing Approaches

| Paper / System | Approach | Result |
|---------------|----------|--------|
| Darwin Gödel Machine | Tree-structured individual evolution | 56.7% SWE-bench |
| OpenHands + GPT-5 | Human-designed framework | 71.8% SWE-bench |
| Aider + GPT-5 | Human-designed framework | 52.0% Polyglot |
| SE-Agent (CAS/Tsinghua) | Three-fold evolution (revise/recombine/refine) | 61.2% success rate |
| LIVE-SWE-AGENT | Live self-evolving on SWE-bench | arXiv 2511.13646 |
